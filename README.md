Grymt Spel
===========

## Prerequisites
* Node.js
* npm - Back end package manager
* MongoDB - Database
* Grunt - Task runner
```
$ sudo npm install -g grunt-cli
```
* Bower - Front end package manager
```
$ sudo npm install -g bower
```
* Build-essential
```
sudo apt-get install build-essential
```

## Installation
```
$ npm install
```

## Run project
```
$ grunt
```
